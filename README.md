# Streaming Finder

Streaming Finder is a web application that helps users discover where to watch movies and series across multiple streaming platforms.

## 🚀 Overview

Users often spend time searching across different streaming services to find where a specific title is available. Streaming Finder centralizes this information into a single, simple interface.

## 🎯 Features (MVP)

- Search movies and series by title
- View matching results
- See detailed information about a title
- Discover on which platforms a title is available
- Redirect to the selected streaming platform

## 🧠 Goal of the Project

This project is designed as a real-world engineering exercise to:

- Build a scalable backend using Java and Spring Boot
- Develop a frontend using Angular
- Apply software architecture principles
- Integrate external data sources
- Practice system design and decision-making
- Use modern development workflows (GitHub, CI/CD, etc.)

## 🏗️ Tech Stack (Initial)

- Backend: Java + Spring Boot
- Frontend: Angular
- Database: PostgreSQL
- Infrastructure: Docker (initially)
